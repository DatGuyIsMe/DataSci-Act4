# IMDB NLP Architectures

A comparative implementation of three distinct natural language processing architectures: Sequence Classification (BERT), Causal Generation (GPT), and Adversarial Generation (Text-GAN).

## Repository Structure
- `/src`: Training and evaluation pipelines.
- `/notebooks`: Exploratory analysis and prototyping.
- `/results`: Metric logs and sample inferences.

## Setup
1. Clone the repository.
2. Run `pip install -r requirements.txt`.
